
# Personal Portfolio Website

This is my personal portfolio website developed using HTML and CSS.
It showcases my skills, education, and projects in a clean and structured format.

## Features
- Responsive layout
- About, Projects, and Contact sections
- Simple and clean UI design

## Technologies Used
- HTML
- CSS

## Purpose
This project was built as a learning exercise to practice front-end development
and create an online presence for future opportunities.
## About

This portfolio website is an initial version created as a foundational project to apply and strengthen my understanding of HTML and CSS. It is intentionally kept simple, focusing on clear structure, layout organization, and basic styling principles.
The project will be continuously improved in the future with additional features, better design modifications, responsiveness, and new projects as I progress in my learning journey.
